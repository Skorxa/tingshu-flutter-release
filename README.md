# 橘听

<p align="center">
  <img src="assets/icon.png" width="160" alt="橘听软件图标" />
</p>

橘听是一款安卓端聚合听书应用。这个仓库只用于公开发布安装包和软件内更新文件，不包含应用源码。

## 下载

最新版本信息：[`version.json`](./version.json)

安装包位于：[`apk/`](./apk/)

软件内更新可以使用下面这个清单地址：

```text
https://raw.githubusercontent.com/Skorxa/tingshu-flutter-release/main/version.json
```

## 内置 GitHub 加速接口

应用内可按顺序尝试以下加速接口，用于改善 GitHub 下载速度：

```text
https://ghfast.top/
https://gh-proxy.com/
https://github.chenc.dev/
https://github.mxw.qzz.io/
https://tvv.tw/
https://gh.927223.xyz/
```

## 仓库说明

- `version.json`：软件内更新清单。
- `apk/`：发布用 APK 文件。
- `assets/icon.png`：橘听软件图标。

源码仓库为私有仓库，发布文件由 GitHub Actions 自动生成并同步到这里。
